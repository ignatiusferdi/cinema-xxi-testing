how to use it

Prerequisites
Download and install Katalon Studio from https://www.katalon.com/
Install Git on your computer from https://git-scm.com/downloads

Clone the repository
Open a terminal or Git Bash on your computer.
Navigate to the directory where you want to clone the repository.
Clone the repository by running the following command:
git clone https://github.com/ignatiusferdi/cinema-xxi-testing.git
The repository will be downloaded to your local computer.

Import the project in Katalon Studio
Open Katalon Studio.
Click on File > Open Project.
Navigate to the directory where you cloned the repository.
Select the folder named cinema-xxi-testing and click OK.
The project will be imported into Katalon Studio.

Run the test suite
Click on the Test Suites folder in the Test Explorer tab.
Double-click on the fulltest test suite.
The test suite will open in the Test Suite Editor tab.
Click on the Run button to execute the test suite.
The test suite will execute and the results will be displayed in the Log Viewer tab.
